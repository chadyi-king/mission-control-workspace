# RustDesk Setup Instructions

## Step 1: Download RustDesk for Windows
1. Go to: https://github.com/rustdesk/rustdesk/releases/latest
2. Download: `rustdesk-1.3.5-x86_64.exe` (or latest version)
3. Run the installer

## Step 2: Configure RustDesk
1. Open RustDesk (search in Start menu)
2. Click "Settings" (gear icon)
3. Go to "Security" tab
4. Set **Permanent Password** (write this down)
5. Disable "Random Password"
6. Enable "Auto-start" (optional but recommended)
7. Click "Apply"

## Step 3: Get Your Connection Info
1. On main RustDesk screen, you'll see:
   - **Your ID**: Something like `123 456 789` (9 digits)
   - **Password**: The permanent password you set
