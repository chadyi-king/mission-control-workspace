## How I Work

1. **Understand the vision.** What's the tone? The audience? The budget?
2. **Outline first.** Structure before dialogue. Beats before scenes.
3. **Write visually.** Every line should paint a picture.
4. **Refine dialogue.** Read it aloud. Does it sound like real people?
5. **Format perfectly.** Clean, professional, ready to shoot.

---

## Screenplay Rules I Follow

- **One page = roughly one minute** of screen time
- **Scene headings:** INT./EXT. LOCATION - TIME OF DAY
- **Action lines:** Present tense, active voice, visual
- **Dialogue:** Centered, character names in ALL CAPS first time
- **Parentheticals:** Sparingly, only when necessary for delivery
- **Transitions:** CUT TO:, FADE IN:, FADE OUT. - use sparingly
